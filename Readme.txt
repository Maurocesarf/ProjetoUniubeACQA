-Tema escolhido: AS CIDADES MAIS BELAS DO MUNDO

Para a realização da Avaliação Continuada de Questão Aberta (ACQA), a proposta consistia na criação de um site em HTML, abordando um tema à escolha do aluno. Este site deve abranger as seguintes considerações:

-Criar uma pasta com meu nome completo, dentro dessa pasta criar uma pasta com o nome imagens, onde todas as imagens usadas serão armazenadas. *OK*
-A página inicial de seu site deverá ter o nome index.html *OK*
-Título da página – Seu RA, nome completo *OK*
-Logotipo da Uniube com link para o site da Uniube abrindo em uma nova aba do navegador *OK*
-Menu, com link para as outras páginas do site *OK*
-Uma página com o nome links.html, você deverá inserir links interessantes relacionados ao assunto do tema escolhido. Estes links deverão abrir em uma nova janela *OK*
-Uma página com o nome imagens.html com diversas imagens sobre seu tema *OK*
-Em alguma página, deverá ser utilizada a estrutura de listas ordenadas e não ordenadas *OK*
-Em alguma página deverá ser utilizada a estrutura de tabelas *OK*

-Página com o nome fale_conosco.html com um formulário de contato com pelo menos as seguintes informações:
> Um campo de texto:
> um campo com a utilização de botões de rádio;
> uma caixa se seleção;
> um textarea:
> botão para envio do formulário.
